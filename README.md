# 🏟️ Sports Hall Booking System (ServiceNow)

## 📖 Overview

The **Sports Hall Booking System** is a ServiceNow application developed to simplify the process of booking sports halls. It enables users to request hall reservations through an easy-to-use interface while allowing administrators to manage bookings efficiently. The system prevents duplicate bookings, automates approval workflows, calculates booking fees, and provides reports and dashboards for monitoring.

---

## 🚀 Features

- Sports Hall Booking
- Record Producer for booking requests

- Business Rules for booking validation
- Client Scripts
- UI Policies
- Service Portal Integration

- Reports and Dashboards
